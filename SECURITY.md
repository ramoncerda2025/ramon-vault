## Políticas de Seguridad 2025-2030 (obligatorias)

| Regla                                    | Detalle                                                      | Cumplimiento |
|------------------------------------------|--------------------------------------------------------------|--------------|
| Repo siempre privado                     | Nunca público, nunca forkable                                | 100%         |
| 2FA obligatorio                          | GitHub + YubiKey + app de respaldo                           | Activado     |
| Contraseñas                              | Bitwarden + mínimo 24 caracteres                             | Siempre      |
| Tokens y claves SSH                      | Vida máxima 90 días, rotación automática                     | Automatizado |
| Revisión de accesos                      | Cada 3 meses (Microsoft, Google, GitHub)                     | Calendario   |
| Backups del vault                        | Repo + copia cifrada local (age + USB)                       | Mensual      |
| Archivo EMERGENCIA.txt                   | Cifrado con GPG, clave con familiares de confianza           | Listo 2026   |
| Nunca guardar contraseñas reales aquí    | Solo pistas o hashes (ej: "primer perro + año nacimiento")   | Estricto     |
